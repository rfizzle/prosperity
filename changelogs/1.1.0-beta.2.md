A small but meaningful beta update focused on container protection consistency and a quality-of-life improvement for players on multiplayer servers.

## Added

- Refreshable containers now stay protected after you've looted them, so your instanced loot is preserved correctly through a container's full refresh cycle.
- When connected to a remote server, Mod Menu now displays the server's Prosperity settings in read-only mode so you can see exactly how the mod is configured without leaving the game.
