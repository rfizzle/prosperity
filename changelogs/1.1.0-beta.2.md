This beta patch tightens up two player-visible behaviours that slipped through in beta.1.

## Added

- Refreshable containers (those that replenish over time) now stay protected between loot cycles — other players can no longer grab your refresh before you return.
- When connected to a remote server, the mod's settings screen in Mod Menu is now shown read-only, reflecting the values the server has configured rather than suggesting you can change them.
